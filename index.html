HASHDUMP

Hashdump is the computer security club at Colorado State University. We participate and put on numerous security competitions every year, and attempt to facilitate a better understanding of security as a whole. All majors are welcome to join and participate.


We meet every Tuesday at 5:00 PM in the Computer Science Building 315. A current schedule is posted below:


Date Talk Notes
03-18 MitM Man in the middle part 2. Learn about wired and wireless Man in the Middle attacks
03-25 Intro to Cryptography Guest talk from an ex-hashdump member about the fundamentals of cryptography and the math that makes our world safe
02-04 Wifi Pineapple An introduction to a nasty little device that facilitates wireless man in the middles
02-18 Malicious Web Code Take a quick look into why web security is so difficult, including path traversal, code injection, and some very basic XSS
02-25 Assembly and Exploitation Our first in depth look at software exploitation and low level assembly.
03-4 XSS Learn the ins and outs of basic XSS and some slightly more advanced attacks
